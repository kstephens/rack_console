# RackConsole

A Rack compatible code introspection and navigation console.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rack_console'
```

## Example Usage

```
bundle
cd example/ex01
bundle exec rackup
```

Browse http://localhost:9292/
